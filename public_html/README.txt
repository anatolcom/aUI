Установка Ruby

Скачать: https://rubyinstaller.org/downloads/

При установке поставить галочки:
- Install Tcl/Tk support
- Add Ruby executable to your PATH
- Assotiate .rb and .rbw files with this Ruby instalation

Запустить 
Start Command Prompt with Ruby
распологается: [USER_HOME]\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Ruby 2.3.3-p222-x64
или в командной строке вызвать: /E:ON /K [RUBY_HOME]\bin\setrbvars.bat

выполнить команду: gem install sass
